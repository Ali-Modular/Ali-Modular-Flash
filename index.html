    How To Update
    
          
      
            
    1. Press and release RESET, then immediately press BOOT within the 3-second timeout — module enters DFU mode.
      2. 
            
    2. Click Connect Device → select DFU in FS Mode from the browser dialog.
      3. 
            
    3. Choose your channel and version, then click Flash Selected. Keep USB connected.
      4. 
            
    4. Wait for the console to confirm completion, then power-cycle the module.
      5. 
          
    
        
    

        
        
    
          
    Troubleshooting
    
          
      
            
    • Windows: Install WinUSB driver via Zadig if the device doesn't appear.
      • 
            
    • Browser: Must use Chrome or Edge on desktop over HTTPS. Safari & Firefox are not supported.
      • 
            
    • Cable: Use a direct USB port and a data-capable (not charge-only) cable.
      • 
            
    • Stuck in DFU? Power-cycle the module and retry from step one.
